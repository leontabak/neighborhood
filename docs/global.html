serverErrorHasBeenReported :boolean

This application requires communication with both the Google Maps server and the Wikipedia server. If either or both of these servers fails to respond to the application's effort to connect, the application will publish a notice, but will publish the advice (e.g., "check your network connection") that is common to both sources of error just once.
